# Eclipse Peon AI Documentation

VitePress-based documentation for Eclipse Peon AI.

## Setup

### Prerequisites
- Node.js 18+
- npm 9+

### Installation

```bash
npm install
```

## Commands

### Development
- `npm run docs:dev` - Start dev server at http://localhost:5173
- `npm run docs:preview` - Preview production build

### Build
- `npm run docs:build` 

### Deployment
- Build output includes `sitemap.xml` for search engines
- All pages maintain original URL structure for SEO
- Pages are tagged for better search engine ranking
